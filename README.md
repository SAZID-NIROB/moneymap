# Expense Tracker MERN

A production-ready MERN expense tracker with JWT authentication, advanced expense filtering, analytics-rich dashboards, dark mode, and a responsive 2026-style UI built with React, Vite, Tailwind CSS, Express, MongoDB, and Recharts.

## Project Structure

```text
client/
  src/
    components/
    context/
    hooks/
    pages/
    services/
server/
  config/
  controllers/
  middleware/
  models/
  routes/
  utils/
```

## Quick Start

1. Copy `server/.env.example` to `server/.env`.
2. Copy `client/.env.example` to `client/.env`.
3. Install dependencies with `npm run install:all`.
4. Start both apps with `npm run dev`.

The API runs on `http://localhost:5000` and the frontend runs on `http://localhost:5173`.
